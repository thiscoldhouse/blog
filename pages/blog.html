If you want to check out the code, you can see it on my github!

As you can probably tell, the site itself is nothing fancy. It consists solely of static files and is written in simple html/css/js, with heavy use of jquery, and hosted on an AWS micro instance using NGINX. I use certbot to manage https. There's really no particular reason for this site to use https, but I support the growing movement for the entire web to be encrypted, and the folks at letsencrypt did a great job with certbot.

It's a single page app. I'm not usually a front-end person, but when I do dabble, I'm always frustrated by front-end frameworks (like angular. Actually, especially angular), so I kept it really simple and rolled out my own super simple routing. Pretty much all of the javascript is in main.js, if you're curious.
